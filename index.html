Conhecimentos

HTML

Estruturar páginas, colocando seus respectivos identificadores e classes, utilizando tags semânticas, tornando o site melhor ranqueável em uma busca orgânica.

CSS 3

Formatação e estilização das páginas HTML, devidamente estruturadas utilizando os seletores de classes e identificadores, pseudoclasses e várias técnicas modernas e avançadas tais como flexbox, grid-layout, responsividade entre outras.

Design

Projeto de páginas web modernas, e visualmente agradáveis,utilizando softwares de edição de imagens e respeitando sempre os vários princípios do design, como utilização mais apropriada das formas, cores, tipografia, imagens etc.

JavaScript

Toda boa página web deve proporcionar um eficaz grau de interatividade a fim de facilitar o usuário na navegação e as diversas tarefas que este necessita realizar em uma web page.

Bootstrap

Para quem domina a estruturação e formatação de páginas web, o framework é uma poderosa aliada para tornar páginas web responsivas com visual moderno, mesmo para quem deseja construí-la do zero.

Git e GitHub

Um dos mais populares softwares de controle de versionamento de projetos nos possibilita melhorar as páginas web, salvando sempre as versões anteriores. Dessa forma o progresso do projeto é otimizado.
